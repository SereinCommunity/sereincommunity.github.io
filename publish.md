# 提交你的作品

我们欢迎每一个用户提交自己的插件/整合包/匹配文件/定时任务文件，为Serein社区添砖加瓦，让Serein变得更加完美✨

::: warning

- 提交并审核通过后不一定马上生效
- 你需要确保你的作品不会违反当地的法律法规

:::
