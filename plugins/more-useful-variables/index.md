## 变量列表

| 变量名                          | 说明             |
| ------------------------------- | ---------------- |
| `deviceinfo_drivetype_{盘符}`   | 驱动器类型       |
| `deviceinfo_driveformat_{盘符}` | 驱动器格式       |
| `deviceinfo_free_{盘符}`        | 驱动器剩余空间   |
| `deviceinfo_used_{盘符}`        | 驱动器已用空间   |
| `deviceinfo_total_{盘符}`       | 驱动器总空间     |
| `deviceinfo_usage_{盘符}`       | 驱动器空间使用率 |
