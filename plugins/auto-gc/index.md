定时调用[`System.GC.Collect()`](https://learn.microsoft.com/zh-cn/dotnet/api/system.gc.collect?view=net-8.0)，减少内存占用

## 食用方法

| 版本  | 操作方法                                       |
| :---: | ---------------------------------------------- |
| v2.0+ | 解压后将**文件夹**复制到`Serein/plugins`下即可 |
| v1.3  | 解压后将 **`gc.js`** 复制到`plugins`下即可     |
